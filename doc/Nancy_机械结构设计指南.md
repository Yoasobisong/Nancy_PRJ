# Nancy 机器人 — 机械结构设计指南（总览）

> 本文档是总览索引。各模块的详细设计请查看 `doc/design/` 下的对应文档。  
> 设计软件：SolidWorks  
> 设计风格：女性瓜子脸 (V 型下颌)，优美拟人化

---

## 设计思路

**由内到外，功能优先：**

```
核心功能机构 (眼球、嘴巴) → 头骨骨架 → 面部硅胶膜 (磁吸) → 化妆上色
      ↑ 先设计                 装得下就行        ↑ 最后做
```

- 内部机构决定"能不能动"
- 骨架决定"装不装得下"
- 面部膜决定"好不好看"
- 三者通过**磁铁**解耦，可独立迭代

## 舵机总表

### 面部舵机 (SCS0009 × 24, 塑料齿轮静音)

| 区域      | 数量 | 驱动方式                  |
| --------- | ---- | ------------------------- |
| 眼球 Pan  | 2    | 连杆拉万向节              |
| 眼球 Tilt | 1-2  | 连杆拉万向节              |
| 上眼睑    | 2    | 推拉弧形滑片              |
| 下眼睑    | 2    | 推拉弧形滑片 (或弹簧被动) |
| 眉毛      | 4    | 连杆拉磁铁                |
| 脸颊      | 2    | 连杆拉磁铁                |
| 上唇      | 3    | 连杆→滑槽→磁铁            |
| 嘴角      | 4    | 双舵机堆叠→磁铁           |
| 下唇      | 2    | 连杆→滑槽→磁铁            |
| 下颌      | 1    | 连杆直驱铰链              |

### 脖子舵机 (SM30BL × 3, 无刷静音)

| 轴    | 运动        | 范围 |
| ----- | ----------- | ---- |
| Yaw   | 转头 (左右) | ±90° |
| Pitch | 点头 (上下) | ±45° |
| Roll  | 歪头 (倾斜) | ±30° |

## SolidWorks 建模顺序

```
Step 1: 舵机简化模型 (STS3032: 24×12×27mm, STS3215: 40×20×40mm)
  ↓
Step 2: 眼球子装配 (Pan/Tilt + 摄像头) → 验证运动范围
  ↓
Step 3: 唇部子装配 (先做一侧的"上唇左"完整链路) → 验证行程
  ↓
Step 4: 下颌 (V型) → 验证开合与连杆
  ↓
Step 5: 将所有子装配排布到空间中 → 确定骨架外形
  ↓
Step 6: 头骨前壳+后壳 (包裹所有机构)
  ↓
Step 7: 脖子 3DOF 子装配
  ↓
Step 8: 总装配 → 干涉检查 → 导出 URDF 到 ROS 仿真
  ↓
Step 9 (最后): 面部皮肤模具
```

## 开发路线

```
SolidWorks 建模 → 导出 URDF/Xacro → ROS Gazebo 仿真
                                        ↓
                                   测试控制代码
                                   (舵机控制 + 表情预设)
                                        ↓
                                   仿真通过 → 3D打印实物
                                        ↓
                                   实物组装 + 调试
                                        ↓
                                   硅胶模具 + 皮肤
                                        ↓
                                   AI 模型部署 (Jetson)
```
