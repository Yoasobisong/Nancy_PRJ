# 模块 1：眼球机构设计

> 每侧 2 DOF (Pan + Tilt) + 内嵌摄像头  
> 优先级：**最高** — 第一个设计的模块

---

## 1. 概述

每个眼球需要 **2 DOF**（水平 Pan + 垂直 Tilt），内部嵌入 **USB 内窥镜微型摄像头**。

DOF 分配：

- **Pan (Yaw)** — 左右看，每侧独立舵机
- **Tilt (Pitch)** — 上下看，可共享 1 个舵机或各自独立

总计：3 个舵机（左 Pan + 右 Pan + 共享 Tilt）或 4 个（各自独立）

### 眼球制作

```
材料分解:

1. 眼球壳体 (Ø26mm 球体)
   ├─ 前半球: SLA 树脂打印 (光滑, 白色)
   │         中心开 Ø8mm 孔 (瞳孔)
   │         粘贴手绘虹膜贴纸或喷涂
   │         嵌入 Ø8mm 亚克力透镜 (保护+聚焦)
   │
   └─ 后半球: FDM PLA 打印 (黑色, 遮光)
              内部有摄像头固定座
              底部开缺口走 USB 线缆
              留出 Pan/Tilt 轴承安装孔
```

## 3. 摄像头选型

| 方案  | 型号                      | 尺寸              | 分辨率 | 接口    | 推荐           |
| ----- | ------------------------- | ----------------- | ------ | ------- | -------------- |
| **A** | OV5647 树莓派摄像头裁剪版 | ~8×8mm            | 5MP    | USB     | ⭐⭐⭐         |
| **B** | OV2640 微型模组           | ~6×6mm            | 2MP    | USB     | ⭐⭐⭐⭐       |
| **C** | **内窥镜摄像头模组**      | **Ø5.5-7mm 圆柱** | 720P   | **USB** | **⭐⭐⭐⭐⭐** |

**推荐方案 C：内窥镜摄像头模组**

- 最小直径 **Ø5.5mm**，完美嵌入 Ø26mm 眼球中心
- 自带 LED（可拆除）
- USB 即插即用，Jetson 直接识别
- 淘宝 ¥10-30

## 4. Pan/Tilt 驱动机构

### 万向节方案（推荐）

```
                 万向节 Pan/Tilt 机构

                     眼球
                    ○○○○○
                   ○○○○○○○
                  ○○○  ◎  ○○○   ← 摄像头在中心
                   ○○○○○○○
                    ○○○○○
                      │
               ┌──────┴──────┐
               │  Tilt 框架   │  ← 3D打印 U型框
               │  ┌────────┐ │
               │  │ Tilt轴 │ │  ← 水平轴，两侧轴承
               │  │ (水平) │ │     眼球绕此轴上下转
               │  └────┬───┘ │
               └───────│─────┘
                       │
                  ┌────┴────┐
                  │ Pan 轴  │  ← 垂直轴
                  │ (垂直)  │     Tilt 框架绕此轴左右转
                  └────┬────┘
                       │
              ┌────────┴────────┐
              │   眼球座 (固定)  │  ← 固定在头骨前壳上
              │   (圆筒形)      │
              └─────────────────┘

    左右看 ← Pan 舵机通过连杆拉 Pan 轴
    上下看 ← Tilt 舵机通过连杆拉 Tilt 轴
```

## 5. 联动控制

- **Pan（左右看）**: 两个眼球各一个舵机，软件联动（便于做聚焦/斜视效果）
- **Tilt（上下看）**: 一个舵机 + 连杆同时驱动两个眼球的 Tilt（或各自独立）

推荐 **3 舵机方案**：左 Pan + 右 Pan + 共享 Tilt
