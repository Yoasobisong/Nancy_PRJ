# 模块 4：眼睑机构设计

> 每侧上眼睑 1 DOF + 下眼睑 1 DOF = 共 4 DOF

---

## 1. 概述

眼睑控制眨眼、闭眼、半闭等表情。每侧 2 个微型舵机。

## 2. 机构原理：弧形滑片式

```
               正视图                      侧视图

    上眼睑 ────══════════── ← 薄片    ┌── 上眼睑 (弧形滑片)
              ╔══════════╗              │   ↕ 上下滑动
              ║          ║           ───┤── 眼球
              ║   眼球   ║              │
              ║          ║              │   ↕ 上下滑动
              ╚══════════╝              └── 下眼睑 (弧形滑片)
    下眼睑 ────══════════── ← 薄片
```

- 上眼睑滑片下滑 → 遮住眼球上部 = 闭眼
- 下眼睑滑片上滑 → 遮住眼球下部
- 上眼睑移动范围 ~12mm，下眼睑 ~5mm

## 3. 驱动方式

```
    舵机摇臂
    │     │
    │  连杆 (推拉)
    │     │
    │  ┌──┴──┐
    │  │滑片 │ ← 弧形薄片，贴合眼球表面弯曲
    │  │     │    沿弧形导轨滑动
    │  └─────┘
    │
    └─ 导轨 (固定在眼球座上)
```

## 4. 材料

- 滑片：**0.5-1mm 白色 POM (赛钢)** 或 **PETG 薄片**，弯成弧形
- 也可做成**硅胶皮肤的一部分**（带磁铁的眼睑形状连接到滑块上）

## 5. 舵机安装

```
    后壳内侧 (俯视)

    ┌────────────────────────────┐
    │  [左上睑舵机]  [右上睑舵机] │  ← STS3032 或 SCS0009
    │       │            │       │
    │    连杆↓         连杆↓     │
    │  ┌────┴────┐  ┌───┴────┐  │
    │  │左上眼睑  │  │右上眼睑 │  │
    │  │滑片     │  │滑片    │  │
    │  └─────────┘  └────────┘  │
    │                            │
    │  (同理下方有下眼睑舵机)     │
    └────────────────────────────┘
```

简化方案：下眼睑可用**弹簧被动复位**，省 2 个舵机。
