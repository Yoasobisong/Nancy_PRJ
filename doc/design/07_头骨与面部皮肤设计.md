# 模块 7：头骨骨架与面部皮肤设计

> 头骨 = 前壳 + 后壳 (3D 打印)  
> 皮肤 = 硅胶膜 (磁吸固定)

---

## 1. 设计顺序

```
模块 1-5 的机构设计完成
     ↓
根据内部机构尺寸 → 设计头骨骨架 (前壳+后壳)
     ↓
根据头骨外形 → 设计面部皮肤模具
     ↓
浇注硅胶 → 上色/化妆
```

**骨架是"装不装得下"的问题，皮肤是"好不好看"的问题 — 完全解耦。**

## 2. 头骨骨架

### 2.1 分区

前壳 + 后壳 + 下颌 三大件（M3 螺丝对接）：

```
    前视图 (瓜子脸)            侧视图

    ┌────────────┐         ┌──────────┐
    │   前壳      │         │  后壳     │
    │ ┌──┐ ┌──┐ │         │ ┌──────┐ │
    │ │眼│ │眼│ │         │ │舵机腔│ │
    │ └──┘ └──┘ │         │ └──────┘ │
    │   鼻       │         ├──────────┤
    ├────╲───────┤         │  前壳    │
    │  下颌╲     │         ├──────────┤
    └─────╲─────┘         │  下颌    │
           V               └──────────┘
```

### 2.2 关键尺寸 (女性瓜子脸)

| 参数     | 尺寸          |
| -------- | ------------- |
| 头部总高 | 210mm         |
| 颧骨宽   | 130mm         |
| 下颌宽   | 95mm          |
| 下巴宽   | 25-30mm (V型) |
| 头部深度 | 175mm         |
| 瞳距     | 60mm          |
| 眼球直径 | 26mm          |
| 嘴宽     | 42-48mm       |

### 2.3 材料

| 部件   | 材料        | 打印参数         |
| ------ | ----------- | ---------------- |
| 前壳   | PLA+ / PETG | 0.2mm, 30% 填充  |
| 后壳   | PLA+ / PETG | 0.2mm, 30% 填充  |
| 下颌   | PETG        | 0.2mm, 40% 填充  |
| 舵机座 | PETG / 尼龙 | 0.15mm, 50% 填充 |

### 2.4 设计要点

1. 前壳：眼球座 (Ø28mm 内径) × 2，嘴巴开口 (~45mm)，磁铁安装槽
2. 后壳：舵机安装架 (24+ 个舵机)，连杆通道，走线孔
3. 下颌：V 型，铰接轴 Ø3mm，下唇磁铁平台
4. 底部法兰：Ø80mm，与脖子 Roll 支架对接
5. 前后壳：4-6 个 M3 螺丝孔

## 3. 面部硅胶皮肤

### 3.1 设计思路

皮肤与骨架完全通过 **磁铁** 连接 — 可随时拆卸更换。

```
    皮肤结构 (剖面)

    外表面 ← 化妆/上色
    ════════════════════ ← 硅胶表皮 2-4mm
    │  磁铁A  │  │  磁铁A  │  ← 嵌在硅胶内侧
    ════════════════════
         ↕ 磁吸       ↕ 磁吸
    │  磁铁B  │  │  磁铁B  │  ← 装在骨架/连杆末端
    ┌────────────────────┐
    │      骨架           │
    └────────────────────┘
```

### 3.2 硅胶选择

| 部位      | 硅胶型号           | 硬度             | 原因         |
| --------- | ------------------ | ---------------- | ------------ |
| 唇部      | **Ecoflex 00-30**  | 极软 Shore 00-30 | 需要大变形   |
| 脸颊/眉毛 | **Dragon Skin 10** | 中软 Shore 10A   | 弹性适中     |
| 额头/侧面 | **Dragon Skin 20** | 中 Shore 20A     | 较少变形区域 |

### 3.3 模具设计

```
制作流程:

1. 3D打印阳模 (头部外形, PLA)
   ↓ 打磨光滑 + 喷底漆
2. 在阳模上浇硅胶做阴模
   ↓ 阴模 = 外表面形状
3. 阳模向内偏移 3mm → 新的"内阳模"
   ↓ 内阳模 = 皮肤内表面
4. 内阳模 + 阴模 → 中间的间隙浇硅胶 → 皮肤
   ↓
5. 脱模 → 在硅胶内侧嵌入磁铁 → 上色/化妆
```

### 3.4 磁铁布局

皮肤上大约需要 **20-30 个磁铁点**：

- 眉毛区域：每侧 2 个 (内+外)
- 眼睑：每侧 2 个 (上+下)
- 脸颊：每侧 1 个
- 上唇：3 个 (左+中+右)
- 下唇：2 个 (左+右)
- 嘴角：每侧 1 个
- 额头/侧面：固定用，每侧 2-3 个

磁铁规格：N52 钕磁铁 Ø4×2mm (骨架侧) ↔ Ø4×2mm (皮肤侧)

### 3.5 化妆/上色

硅胶上色方法：

1. **硅胶专用颜料** (Silc Pig 等) 混入硅胶浇注 → 基础肤色
2. **干粉/气垫粉底** → 细节肤色层次
3. **硅胶面漆** (Psycho Paint) → 透明保护层
4. 眉毛/睫毛 → 植入或粘贴人造毛发

> 化妆是纯美术技能，可以多次练习，皮肤做坏了重新浇注即可。
