# 模块 5：眉毛与脸颊机构设计

> 眉毛：每侧 2 DOF (内侧+外侧) = 共 4 DOF  
> 脸颊：每侧 1 DOF (上提) = 共 2 DOF  
> 合计 6 DOF

---

## 1. 眉毛机构

### 1.1 概述

每侧眉毛 2 个牵引点（内侧 + 外侧），通过磁铁拉动硅胶皮肤。

### 1.2 机构原理

```
    眉毛机构 (正面视图, 右侧)

                            ← 眉毛外侧
    ┌─── 内侧牵引点 ──── 外侧牵引点 ───┐
    │         ↕                ↕        │ ← 硅胶皮肤
    │     磁铁A1           磁铁A2       │
    ═════════════════════════════════════
                ↕ 磁吸          ↕ 磁吸
            磁铁B1          磁铁B2
                │               │
         连杆1 ↕          连杆2 ↕
            ┌───┴───┐      ┌───┴───┐
            │舵机1  │      │舵机2  │
            │(内侧) │      │(外侧) │
            └───────┘      └───────┘

    控制效果:
    - 舵机1+2 同时上 → 抬眉 (惊讶)
    - 舵机1+2 同时下 → 皱眉 (生气)
    - 舵机1上 + 舵机2下 → 内侧上扬 (忧虑)
    - 舵机1下 + 舵机2上 → 内侧皱起 (愤怒)
```

**简化方案**：每侧 1 DOF，省 2 个舵机。

### 1.3 舵机安装

安装在**头骨后壳顶部**，长连杆绕过头骨顶部向前到达眉毛位置。

```
    头骨侧视图

         ┌──── 头顶 ────┐
         │   连杆通道    │
    后壳 │  ┌─────────┐ │ 前壳
         │  │ 连杆     │ │
    ┌────┤  │ (弯曲)  │ │ 眉毛位置
    │舵机│→ │→ → → → →│→│→ 磁铁 → 皮肤
    └────┘  └─────────┘ │
         │              │
```

## 2. 脸颊机构

### 2.1 概述

每侧 1 DOF — 上提脸颊 = 笑肌效果（苹果肌）。

### 2.2 机构

```
    脸颊控制 (正面)

    ┌─────────────────────┐
    │        脸颊         │
    │    磁铁A ○──── ← 硅胶皮肤
    │         ↕ 磁吸      │
    │    磁铁B ○          │
    │         │ 连杆      │
    │    ┌────┴────┐      │
    │    │  舵机   │ ← 安装在嘴角舵机附近
    │    └─────────┘      │
    └─────────────────────┘

    效果: 舵机拉 → 脸颊上提 → 苹果肌凸起 → 笑
```
