# 模块 2：下颌与嘴部机构设计

> 下颌开合 1 DOF  
> 优先级：**高** — 第二个设计的模块

---

## 1. 概述

下颌 1 DOF 控制嘴巴张合。瓜子脸设计中下颌是 **V 型收窄**的，下巴尖处仅 25-30mm 宽。

## 2. 机构原理

```
    侧视图

         ┌───── 上颌 (头骨前壳的一部分, 固定) ─────┐
         │                                          │
    铰接轴 ◎ ←── 下颌绕此轴旋转                     │
         │                                          │
         └───── 下颌 (可旋转件, V型) ──────────────┘
                  │
                  │ 连杆/直驱
                  │
            ┌─────┴─────┐
            │ 下颌舵机   │  ← STS3032，安装在后壳/颈部
            └───────────┘
```

**铰接轴位置**：约在耳朵对应处（两侧各一个轴孔），模拟颞下颌关节。

**活动范围**：0°（闭合）~ 25-30°（最大张开）

## 3. 瓜子脸下颌特殊设计

```
    正面视图 (下颌零件)

    ┌──────────────────┐
    │  铰接轴孔 ◎    ◎ │  ← 两侧轴孔 Ø3mm
    │                  │
    │  ← 95mm 宽 →     │  ← 下颌角处
    │    ╲          ╱   │
    │     ╲        ╱    │  ← V 型收窄
    │      ╲      ╱     │
    │       ╲    ╱      │
    │        ╲  ╱       │
    │         ╲╱        │  ← 下巴尖 25-30mm 宽
    └──────────────────┘

    前面做成平台形状 → 用于安装下唇硅胶和磁铁
```

## 4. SolidWorks 设计要点

1. 下颌是独立零件，通过 **Ø3mm 不锈钢轴**与头骨前壳铰接
2. 两侧留轴孔（压入微型轴承 Ø3×Ø6×2.5mm 或直接轴孔 + 铜套）
3. 舵机连杆连接在下颌内侧的连接点上
4. 下颌前面做成**平台形状**，用于安装下唇的硅胶和磁铁
5. **外轮廓为 V 型**，符合瓜子脸

## 5. 关键零件

| 零件     | 尺寸              | 材料           |
| -------- | ----------------- | -------------- |
| 下颌本体 | ~95×70×30mm (V型) | PETG, 40% 填充 |
| 铰接轴   | Ø3mm × 15mm       | 不锈钢         |
| 微型轴承 | Ø3×Ø6×2.5mm       | 标准件         |
| 驱动连杆 | Ø2mm × 30-40mm    | 碳纤棒/钢丝    |
| 舵机     | STS3032           | -              |

## 6. SolidWorks 零件清单

```
Jaw/

```
