# 模块 3：唇部机构设计

> 10 DOF — 最复杂的面部区域  
> 上唇(3) + 下唇(2) + 嘴角(4) + 下颌(1, 见模块 2)

---

## 1. 概述

唇部是论文中 Wav2Lip 唇同步的核心执行器。10 个 DOF 分布在嘴唇周围：

```
         ┌── 上唇左 (1DOF ↕)
         │
上唇中 ──┼── 上唇中 (1DOF ↕)
         │
         └── 上唇右 (1DOF ↕)

嘴角左 ──────( 嘴巴 ~45mm 宽 )────── 嘴角右
(2DOF:                                (2DOF:
 前后+上下)                            前后+上下)

         ┌── 下唇左 (1DOF ↕)
         │
         └── 下唇右 (1DOF ↕)

         下颌开合 (1DOF) → 见模块 2
```

## 2. 推拉连杆机构

每个唇部控制点的链路：

```
舵机 → 摇臂 → 连杆 → 滑块/导向槽 → 磁铁持架 → 磁铁
                                                    ↕ 磁吸
                                                  硅胶皮肤内侧磁铁

详细侧视图:

┌──────┐
│ 舵机 │ ← 固定在头骨上
│ STS  │
│ 3032 │
└──┬───┘
   │ 舵机摇臂 (输出轴)
   │  ┌─┐
   └──┤ ├─── 连杆 (碳纤棒 Ø2mm, 长 20-40mm)
      └─┘     │
              │
       ┌──────┴──────┐
       │  导向滑槽    │ ← 3D打印，约束运动为直线
       └──────┬──────┘
              │
       ┌──────┴──────┐
       │  磁铁持架    │ ← 末端磁铁 N52, Ø4×2mm
       │  ┌──────┐   │
       │  │磁铁B │   │ ← 骨架侧
       │  └──┬───┘   │
       └─────│───────┘
             │ ↕ 磁吸力 (~0.5-1N)
       ┌─────┴───────┐
       │  │磁铁A │   │ ← 硅胶侧
       └─────────────┘
       ═══════════════ ← 硅胶皮肤
```

## 3. 舵机布局（瓜子脸约束）

> ⚠️ 嘴宽仅 ~45mm，下颌 95mm → 嘴两侧仅各 ~25mm 空间

```
            正面视图 (窄嘴布局)

      ┌─── 前壳 (颧骨宽 130mm) ───┐
      │                            │
      │  ┌──┐          ┌──┐       │
      │  │S1│ 上唇左   │S6│ 上唇右│
      │  └──┘          └──┘       │
      │     ┌──┐                  │
      │     │S5│ 上唇中 (鼻下)    │
      │     └──┘                  │
      │  ┌──┐          ┌──┐       │
      │  │S2│ 嘴角左1  │S7│ 嘴角右1│  ← ⚠️ 紧贴嘴巴
      │  │S3│ 嘴角左2  │S8│ 嘴角右2│  ← 竖直堆叠！
      │  └──┘          └──┘       │
      │  ┌──┐          ┌──┐       │
      │  │S4│ 下唇左   │S9│ 下唇右│
      │  └──┘          └──┘       │
      └────────╲────╱─────────────┘
                V 型下颌

    STS3032 尺寸: 24×12×27mm → 竖放时占 12mm 宽
    嘴侧可用空间 ~25mm → 放 2 个竖放舵机刚好

    备选：舵机后移到后壳，用长连杆前伸
```

## 4. 嘴角双舵机堆叠

每侧嘴角 **2 个舵机正交安装**，实现 2D 平面运动：

```
    嘴角机构 (侧视, 右侧)

                     ← 前后方向 →
                  ┌─────────────┐
                  │ 嘴角磁铁持架 │ ← 磁吸硅胶皮肤
                  └──────┬──────┘
                         │
              ┌──────────┴──────────┐
              │   滑块 (上下方向)    │
              └──────────┬──────────┘
                         │ 连杆2
              ┌──────────┴──────────┐
              │  舵机2 (控制上下)    │
              └──────────┬──────────┘
              ┌──────────┴──────────┐
              │   滑块 (前后方向)    │
              └──────────┬──────────┘
                         │ 连杆1
              ┌──────────┴──────────┐
              │  舵机1 (控制前后)    │
              └─────────────────────┘

    效果:
    - 舵机1 → 嘴角前突(嘟嘴)/回缩(微笑)
    - 舵机2 → 嘴角上提(微笑)/下拉(悲伤)
```

## 5. 唇部欠驱动翻转

论文关键特性：唇部连接器带 **被动旋转自由度**。

```
    正常:              下推时自动翻转:

    ═══════            ═══════
    │磁铁│              ╲磁铁╲  ← 唇部外翻！
    ├────┤               ╲────╲   模仿 "u", "o" 音
    │铰链│ ← 弹簧复位    │铰链│
    ├────┤               ├────┤
    │连杆│ ↓ 下推        │连杆│ ↓ 下推
    └────┘               └────┘

    实现: 磁铁持架通过小铰链与连杆末端连接
          铰链有轻微扭簧预紧
```
