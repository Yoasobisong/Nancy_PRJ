# Nancy 仿人机器人脸部 — 项目规划

> 目标：复刻 Columbia 大学 Emo/Lip-Sync 机器人的脸部表情与唇部同步能力  
> 外观：**女性瓜子脸 (V 型下颌)**，优美拟人化，精致五官  
> 脖子：**3DOF 串联结构** (Yaw + Pitch + Roll)  
> 自行设计结构件（SolidWorks → 3D 打印 + CNC）  
> 开发路线：SolidWorks → URDF → ROS 仿真 → 实物

### 2.2 唇部机构详细设计（重点！10 DOF）

```
                ┌─ 上唇左 (1DOF: ↕ + 被动翻转)
                │
     上唇中 ────┤─ 上唇中 (1DOF: ↕)
    (1DOF: ↕)   │
                └─ 上唇右 (1DOF: ↕ + 被动翻转)

  嘴角左 ─────────────────────────────── 嘴角右
  (2DOF:          开口区域              (2DOF:
   回缩↔前突                              回缩↔前突
   上↔下)                                 上↔下)

                ┌─ 下唇左 (1DOF: ↕ + 被动翻转)
                │
                └─ 下唇右 (1DOF: ↕ + 被动翻转)

                  下颌 (1DOF: 张合)
```

**关键设计要点：**

1. **嘴角双电机堆叠**：每侧 2 个舵机正交安装，一个控制回缩/前突，一个控制上提/下拉
2. **上/下唇连接器带欠驱动旋转轴**：
   - 上唇下降时，连接器自动向外翻转 → 模仿撅嘴 (w/r/u 音)
   - 下唇上升时，同理外翻
   - 使用被动弹簧/柔性铰链实现
3. **推+拉双向驱动**：连杆机构可以推硅胶皮肤也可以拉，比线缆好
4. **磁性连接器**：舵机连杆末端 → 磁铁 → 磁吸到硅胶皮肤内侧的磁铁

### 2.3 磁性快拆连接器设计

```
      硅胶皮肤(外)
    ═══════════════
    │  ┌───────┐  │
    │  │ 磁铁A │  │  ← 强力胶粘在硅胶皮肤内侧
    │  └───┬───┘  │
    ═══════╪══════
           ↕       ← 磁吸力连接 (可快速拆装)
    ┌──────┴──────┐
    │   磁铁B     │  ← 固定在机械持架上
    │   (持架)    │
    └──────┬──────┘
           │
    ┌──────┴──────┐
    │  连杆/舵机   │  ← 3D打印连杆
    └─────────────┘
```

**磁铁规格建议：**

- 钕磁铁 (N52)，Ø4mm × 2mm 圆柱形
- 每个连接点一对（皮肤侧 + 持架侧）
- 总计约 10-12 对（唇部区域）+ 8-10 对（脸颊/眉毛区域）

### 3.1 为什么不用 Stewart？

|          | Stewart (旧方案) | **串联 3DOF (新方案)**         |
| -------- | ---------------- | ------------------------------ |
| 运动范围 | 小 (±15°)        | **大 (±45-90°)**               |
| 外观     | ❌ 不像脖子      | ✅ 自然                        |
| 复杂度   | 很高             | **低**                         |
| 舵机数   | 6                | **3**                          |
| ROS 仿真 | 复杂 (闭链)      | **简单 (URDF revolute joint)** |

## 四、舵机选型（静音方案）

> ⚠️ 噪声控制是仿人机器人的关键。全部选用静音舵机。

### 4.1 面部舵机 — Feetech SCS0009 × 24

| 参数     | 规格                      |
| -------- | ------------------------- |
| **型号** | Feetech SCS0009           |
| **扭矩** | 1.5kg·cm                  |
| **尺寸** | 23×12×29mm                |
| **齿轮** | **塑料齿轮** (静音关键！) |
| **通信** | SCS TTL 串行总线 (半双工) |
| **反馈** | 位置/温度/电压            |
| **价格** | ¥15-25                    |

**为什么选 SCS0009？**

- ✅ **塑料齿轮 = 安静**（金属齿轮的 STS3032 噪声 ~70dB，塑料齿轮 ~50dB）
- ✅ 1.5kg·cm 扭矩足够拉硅胶皮肤 (实际负载 < 0.3kg·cm)
- ✅ 尺寸紧凑 23×12mm
- ✅ SCS 串行总线，一条线串联所有面部舵机
- ✅ 便宜 (¥20/个)，24 个才 ¥480
- ⚠️ 塑料齿轮耐久性稍差，但面部运动幅度小，可接受

### 4.2 脖子舵机 — Feetech SM30BL × 3

| 参数     | 规格                        |
| -------- | --------------------------- |
| **型号** | Feetech SM30BL              |
| **扭矩** | 30kg·cm                     |
| **电机** | **无刷电机** (静音关键！)   |
| **齿轮** | 钢齿轮 + 铝合金 CNC 外壳    |
| **通信** | SMS RS485 串行总线 (半双工) |
| **反馈** | 位置/速度/温度/电流/负载    |
| **尺寸** | 40×28×39mm                  |
| **价格** | ¥150                        |

**为什么选 SM30BL？**

- ✅ **无刷电机 = 极安静**（无碳刷摩擦噪声）
- ✅ 30kg·cm 大扭矩，承载头部 (~300g) 绰绰有余
- ✅ RS485 通信，抗干扰、传输距离长
- ✅ 12-bit 磁编码器，0.088° 精度

> 面部总线 (SCS TTL) 和脖子总线 (SMS RS485) 协议不同，**各走一路 UART**，互不干扰。

## 五、硅胶面部皮肤制作

### 5.1 制作流程

```
Step 1: 面部3D建模
  │  在 Blender/Fusion360 中建模人脸外形
  │  导出 STL
  ↓
Step 2: 制作阳模 (Positive Mold)
  │  3D打印人脸的内表面形状
  │  包含磁铁安装凹槽和舵机连接点
  │  打磨光滑
  ↓
Step 3: 制作阴模 (Negative Mold)
  │  3D打印人脸的外表面形状
  │  或用翻模硅胶(Mold Star 15 SLOW)制作
  │  喷脱模剂
  ↓
Step 4: 浇注硅胶
  │  混合硅胶 (Ecoflex 00-30 或 Dragon Skin 10)
  │  真空脱泡
  │  注入阳模和阴模之间的空隙
  │  室温固化 4-6 小时
  ↓
Step 5: 脱模 + 安装磁铁
  │  撕开阴模取出硅胶面部
  │  在预留位置用硅胶粘合剂粘入磁铁
  ↓
Step 6: 上色
  │  使用硅胶颜料 (Silc Pig) 或气笔喷涂
  │  分层上色：底色 → 血色 → 细节
```

## 六、电子系统架构

> **计算平台**: Jetson Orin Nano Super 8GB (67 TOPS, ~¥1900)  
> **舵机控制**: Jetson 直接通过 UART 驱动 STS/SCS 串行总线舵机，**无需 MCU**  
> **传感器**: 仅 USB 摄像头 ×2 + 麦克风 + 扬声器（论文所需的全部传感器）  
> **AI 策略**: 本地跑全部视觉模型 (PyTorch/CUDA 原生)，LLM 可本地小模型或 API

### 6.1 为什么不需要 MCU？

STS/SCS 系列舵机是**串行总线舵机**，直接用半双工 UART 即可控制：

- Jetson 有多路 UART，直接发送 STS/SCS 协议指令
- 只需一个 **74HC126 三态缓冲器** (~¥3) 做半双工收发转换
- Python `pyserial` 库即可完成所有通信
- 不需要自定义协议，STS/SCS 本身就有完整的协议（读写位置/速度/温度等）

### 6.2 为什么不需要额外传感器？

复刻论文所需的传感器**只有 3 种**，全部通过 USB 连接 Jetson：

| 传感器                     | 用途 (论文中)                     | 接口     |
| -------------------------- | --------------------------------- | -------- |
| **USB 摄像头 ×2** (眼球内) | 观察人脸表情 + 自建模时录自己的脸 | USB      |
| **麦克风**                 | 接收语音 → Wav2Lip 生成口型       | USB 声卡 |
| **扬声器**                 | 播放 TTS 语音                     | USB 声卡 |

❌ 不需要 IMU、力传感器、温度传感器、距离传感器等。

### 6.5 CUDA 模型部署（零转换！）

论文代码全是 PyTorch → Jetson 上 **直接运行**，无需任何模型格式转换：

```

可选加速 (后期优化):
PyTorch → TorchScript → TensorRT
性能提升 3-5x，但不是必须的
```

## 八、开发阶段规划

### Phase 0: 准备 (1-2 周)

- [ ] 确定最终 DOF 数量和舵机型号
- [ ] 采购舵机、硅胶材料、磁铁、球铰等
- [ ] 准备 3D 打印机 (FDM + SLA 如果可能)
- [ ] 搭建开发环境 (Python + PyTorch + OpenCV)

- [ ] 设计 Stewart 平台 CAD (Fusion 360 / SolidWorks)
- [ ] 3D 打印上下平台 + 舵机安装座
- [ ] 组装 Stewart 平台
- [ ] 编写逆运动学算法
- [ ] 测试基本运动 (点头/摇头/歪头)

### Phase 2: 面部骨架 + 唇部机构 (3-4 周)

- [ ] 面部骨架 CAD 设计（头骨、眼球座、舵机安装位）
- [ ] 唇部 10 DOF 连杆机构设计
- [ ] 嘴角双电机堆叠机构设计
- [ ] 上/下唇欠驱动翻转机构设计
- [ ] 磁性连接器持架设计
- [ ] 3D 打印 + 组装
- [ ] 眉毛/眼睑机构设计安装

### Phase 3: 硅胶面部皮肤 (2-3 周)

- [ ] 面部外形 3D 建模（美术方向）
- [ ] 阳模 + 阴模 3D 打印
- [ ] 浇注硅胶 (多次迭代！)
- [ ] 磁铁安装
- [ ] 上色

- [ ] USB 摄像头 ×2 + USB 声卡 集成测试
- [ ] 电源系统搭建 (12V→5V BEC + Jetson 供电)
